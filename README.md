# Syntexhub_Password_Strength_Checker
A beginner-friendly cybersecurity mini project built in Python to analyze password strength based on length, characters, digits, and symbols. The tool classifies passwords as WEAK, MEDIUM, or STRONG and provides security improvement suggestions.
